# 🚀 Como Contribuir

1. **Task Driven:** Sempre escolha uma task do `BACKLOG.md` **enquanto não crio as ISSUES no github.**
2. **Feature Branch:** `git checkout -b nome-da-feature`.
3. **Commit:** O Husky rodará o Lint. Se falhar, corrija antes de subir.
4. **Deploy:** A Vercel gerará uma Preview URL para validação.
